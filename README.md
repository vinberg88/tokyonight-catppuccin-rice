# tokyonight-catppuccin-rice

A GTK theme based on the colours of Folke's great Tokyonight theme for Neovim,
VinceLiuice's awesome GTK themes, and the creativity of Gusbemacbe's Suru Plus
Icon Theme.

How to install script for Cinnamon Desktop and UBUNTU 26.04

```bash
bash <(curl -sL https://raw.githubusercontent.com/vinberg88/tokyonight-catppuccin-rice/main/installer.sh)
```

You will install this from script
